filebeat 部署

原创
2017-07-16
elk
filebeat

  • rpm方式安装
    rpm -vi filebeat-5.3.0-x86_64.rpm

  • 配置filebeat.yml
    vi /etc/filebeat/filebeat.yml

  • 启动命令
    sudo /etc/init.d/filebeat start|stop|status
  • 自启动
    systemctl enable filebeat.service
