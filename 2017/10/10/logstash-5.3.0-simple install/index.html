制作logstash 一键安装脚本

原创
2017-10-10
elk
logstash

简介

默认制作的脚本安装至/usr/local/logstash_server/logstash-5.3.0下,版本为：5.3.0

默认的java安装目录为：/usr/local/java

制作步骤

  • 1.准备logstash-5.3.0.tar.gz原装包
  • 2.解压至logstash_server下
  • 3.复制脚本tail_start.sh|tail_stop.sh至logstash_server/logstash-5.3.0下
  • 4.解压复制附件中的config目录至对应logstash_server/logstash-5.3.0/config目录
  • 5.压缩 tar -czvf logstash_server.tar.gz logstash_server
  • 6.放置logstash_server.tar.gz和logstash_install.sh同级至任何目录执行
  • 7.安装命令：sh logstash_install.sh -h 127.0.0.1:9200
    -h 为es的http 地址
